**RolAsyst™** to wolne narzędzie stworzone z myślą o szybkim szacowaniu czasu pracy oraz zapotrzebowania na paliwo podczas obrabiania pól uprawnych o określonym metrażu (w hektarach).

---

## 🚜 Główne funkcje

* **Precyzyjne obliczenia:** Przelicza powierzchnię pola (ha) na szacowane zużycie paliwa oraz czas pracy.
* **Generowanie raportów:** Tworzy przejrzyste podsumowanie wyników.
* **Zapis do pliku:** Możliwość szybkiego wyeksportowania raportu tekstowego na dysk (wystarczy podać nazwę pliku).
* **W pełni Wolne Oprogramowanie:** 100% otwartego kodu i etycznej wolności.

---

## 🚀 Szybki start / Binarki

Program jest dostępny w gotowej wersji binarnej na systemy **Linux®**, **Amiga® (w przygotowaniu)**, **FreeDOS®**, **macOS® (w przygotowaniu)** **Windows® (ale jego lepiej pominąć)**, więc zadziała nawet na systemie operacyjnym klamki do drzwi.

RolAsyst stawia na wolność i wieloplatformowość.

## 🛠️ Zbuduj ze źródeł

1. Sklonuj repozytorium:
   ```bash
   git clone https://github.com/alankingpl0-oss/rolasyst.git
   cd rolasyst
   fpc gospo
   ```

   Ta komenda nie rozwali systemu plików, jednak stworzy kilka nowych i-węzłów, które później będziesz mógł bezpowrotnie usunąć.
